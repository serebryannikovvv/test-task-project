# Liquid Sort Solver

Решение головоломки "Сортировка жидкостей" на Java 8.

## Запуск

```bash
# Сборка и запуск (по умолчанию алгоритм BFS)
./gradlew run

# Сборка и запуск (алгоритм DFS)
./gradlew runBFS

# Сборка и запуск (алгоритм DFS)
./gradlew runDFS

# Без анимации
./gradlew build run --args="--no-animation"
